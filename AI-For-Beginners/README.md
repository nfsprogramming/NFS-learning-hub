## 🛠️ Quick Start

### 🌐 Online Access
The portal is optimized for Vercel deployment and deep-linking support.

### 💻 Local Development
If you want to run the portal on your own computer:
1.  **Clone the Repo**: `git clone https://github.com/nfsprogramming/Microsoft-Course.git`
2.  **Launch Dashboard**: Double-click `Launch_Dashboard.bat` (Windows) or run `python run_portal.py`.
3.  **Explore**: The portal will automatically open in your browser at `http://localhost:8000`.
